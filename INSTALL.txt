先决条件：

    for windows: download and install python3,pip3 package 
 
    for linux: sudo apt install python3 python3-tk python3-pip

Winodws下使用：

    1.假设工具路径为：D:\\Temperature-tools

    2.添加环境变量：PATH=%PATH%;D:\\Temperature-tools
    
    3.打开cmd，运行命令fitx ok，安装工具需求

Linux下使用：

    1.假设工具路径为：/opt/Temperature-tools

    2.添加环境变量：PATH=$PATH:/opt/Temperature-tools
 
    3.同上，终端运行fitx ok

工具卸载：

    1.运行命令fitx bye

    2.清除安装时添加的环境变量

    3.删除工具目录
